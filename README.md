# Techometer for E-Bike

Basic overview of the tachometer controlled by arduino nano.

## Connections

Hardware used:
- Arduino Nano (Atmega328p)
- SSD1306 (182x64) Display


	Display		Arduino

	GND  -->  GND<br>
	VDD  -->  3v3<br>
	SCK  -->  SCK (D13)<br>
	SDA  -->  MOSI (D12)<br>
	RES  -->  SS (D10)<br>
	DC   -->  D9<br>
	CS   -->  D8<br>
